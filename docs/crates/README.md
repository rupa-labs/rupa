# Crate Guide (Tier 1) 🧱

Detailed 1:1 documentation for every Atomic Material in the Rupa Framework. These crates form the foundational "DNA" and "Ports" of the system.

---

## 🧬 The DNA (Core Primimitives)
*   **[rupa-signals](./rupa-signals.md)**: Fine-grained reactivity engine.
*   **[rupa-vnode](./rupa-vnode.md)**: Universal UI contract and Style models.
*   **[rupa-base](./rupa-base.md)**: Foundational types (Color, Vec2, Rect, Id).
*   **[rupa-motion](./rupa-motion.md)**: Animation and spring physics engine.

---

## 🔌 The Ports (Infrastructure)
*   **[rupa-context](./rupa-context.md)**: Reactive Dependency Injection.
*   **[rupa-auth](./rupa-auth.md)**: Agnostic Identity & Session management.
*   **[rupa-assets](./rupa-assets.md)**: Resource loading and caching system.
*   **[rupa-i18n](./rupa-i18n.md)**: Reactive Internationalization.
*   **[rupa-net](./rupa-net.md)**: Reactive Network and I/O.
*   **[rupa-store](./rupa-store.md)**: Persistent state management.
*   **[rupa-broadcast](./rupa-broadcast.md)**: Global reactive event bus.
*   **[rupa-queue](./rupa-queue.md)**: Asynchronous task orchestration.

---

## ⚙️ Systems & Logic
*   **[rupa-forms](./rupa-forms.md)**: Form objects and validation engine.
*   **[rupa-telemetry](./rupa-telemetry.md)**: Observability (Logs, Metrics, Profiling).
*   **[rupa-a11y](./rupa-a11y.md)**: Accessibility semantics and OS bridging.
*   **[rupa-console](./rupa-console.md)**: Low-level terminal utilities.
*   **[rupa-canvas](./rupa-canvas.md)**: High-performance low-level drawing.
*   **[rupa-test](./rupa-test.md)**: Headless testing and snapshot utilities.
