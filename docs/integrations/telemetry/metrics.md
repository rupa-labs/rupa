# Module: Performance Metrics (`crates/rupa-telemetry/src/metrics.rs`) 📊

This module collects real-time performance data from the framework.

---

## 🏗️ Data Points
- **FPS**: Frames per second.
- **Render Time**: Duration of the `render()` phase.
- **Patch Count**: Number of atomic instructions sent to the GPU in the last frame.
- **Signal Activity**: Tracks the frequency of state mutations.
