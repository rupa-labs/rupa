# Doc Orchestrator Architecture 📚

The **Doc Orchestrator** (the `rupa-docs` crate) is an interactive documentation viewer built using the Rupa Framework itself.

---

## 1. Purpose

It serves as a live demonstration of the framework's capabilities, capable of running on Desktop, Web, and Terminal.

---

## 2. Technical Composition

Uses the **Markdown Engine** to parse the workshop documentation and renders it using **Standard Layouts** and **Agnostic UI** components.
