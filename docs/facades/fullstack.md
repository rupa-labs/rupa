# Artisan Showroom: Rupa Full-Stack (`crates/rupa-fullstack`) 🌌

The universal showroom for building synchronized multi-platform ecosystems. It bundles everything needed for high-end Web + Server + Data integration.

---

## 🏗️ Pre-composed Bundle
- **Tiers**: All Atomic Materials + All Composite Assemblies
- **Web**: `rupa-web-core`, `rupa-router`
- **Server**: `rupa-server-core`, `rupa-auth`
- **Infrastructure**: `rupa-store`, `rupa-net`, `rupa-i18n`

## 🏁 Prelude
Provides the most comprehensive Rupa experience, enabling isomorphic Rust code that runs on both client and server.
