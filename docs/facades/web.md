# Artisan Showroom: Rupa Web (`crates/rupa-web`) 🌐

The entry point for building reactive, WASM-based frontend applications.

---

## 🏗️ Pre-composed Bundle
- **Logic**: `rupa-core`, `rupa-signals`
- **Visuals**: `rupa-ui` (UI Component System)
- **Execution**: `rupa-web-core` (WASM Hydration)
- **Infrastructure**: `rupa-router`, `rupa-auth`, `rupa-store` (WebStorage)

## 🏁 Prelude
Provides a web-optimized prelude:
- DOM-agnostic components
- Browser routing hooks
- Authentication context providers
