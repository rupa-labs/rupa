# Module: Internationalization System (`crates/rupa-i18n`) 🌍

The `rupa-i18n` crate handles localization, dictionary management, and culturally aware formatting.

---

## 1. Core Philosophy: Reactive Dictionaries
Language selection is a `Signal`. Changing the language instantly updates all translated text nodes in the VNode tree without requiring an application reload.

## 2. Key Features
- **Dictionary Loading**: Fetch JSON/Fluent files locally or via network.
- **Formatting**: Dates, Numbers, and Currencies based on the user's Locale.
- **Fallback**: Graceful degradation to a default language if a key is missing.

## 3. Key Exports (1:1 Mapping)
- `I18nProvider`: The Context provider holding the current language signal.
- `t!(key, args)`: The macro/function to resolve translation strings reactively.
- `Locale`: Struct representing the active region settings.
