# Path: Mobile Mobility (Mobile Adapter) 📱

The **Mobile Mobility** template is for Artisans building native experiences for the palm of the hand. It bridges the Rupa Kernel to mobile-specific hardware and touch-first interactions.

---

## 🏗️ What's Inside?

- **Mobile Adapter**: Pre-configured `rupa-mobile` (Vulkan for Android, Metal for iOS).
- **SafeArea Management**: Automatic handling of notches, status bars, and home indicators.
- **Touch Intent Mapping**: Specialized gestures (Swipe, Pinch, Long Press) mapped to Rupa intents.

---

## 🎯 Best For:

1.  **Cross-platform Mobile Apps**: Sharing 90% of UI logic between iOS and Android.
2.  **Performance-critical Mobile UI**: Leveraging low-level GPU access for smooth transitions.
3.  **Artisan Portfolios**: Showcasing your work on mobile devices with high aesthetic integrity.

---

## 🚀 Quick Start

Initialize via CLI:
```bash
rupa create --template mobile
```

### Environment Note
Requires Android NDK for Android or Xcode for iOS. See [**Installation**](../installation.md) for setup.
