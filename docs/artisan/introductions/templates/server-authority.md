# Path: Server Authority (Server Adapter) 🛡️

The **Server Authority** template is engineered for the modern web's backbone. It focuses on Server-Side Rendering (SSR) to deliver lightning-fast initial loads and perfect SEO.

---

## 🏗️ What's Inside?

- **Server Adapter**: Pre-configured `rupa-server` for HTML streaming.
- **HTML Serialization**: High-performance VNode-to-HTML conversion engine.
- **Hydration Ready**: Patterns for delivering state from server to client-side Rupa apps.

---

## 🎯 Best For:

1.  **SEO-driven Websites**: Marketing pages, blogs, and public dashboards.
2.  **Full-stack Rupa Apps**: Using the same components for both server rendering and client interaction.
3.  **High-performance APIs**: Serving reactive UI fragments over the wire.

---

## 🚀 Quick Start

Initialize via CLI:
```bash
rupa create --template server
```
