# Path: Terminal Arts (Terminal Adapter) 📟

For those who live in the command line, the **Terminal Arts** template provides a reactive, aesthetic TUI engine that makes CLIs feel like modern applications.

---

## 🏗️ What's Inside?

- **Terminal Adapter**: Pre-configured `rupa-terminal` (Crossterm).
- **TUI Orchestrator**: High-level character-grid layout management (`rupa-tui`).
- **Console Port**: Low-level ANSI materials and CLI primitives (`rupa-console`).
- **Standard TUI Components**: Panels, Lists, and ASCII-aware layouts.

---

## 🎯 Best For:

1.  **DevOps Utilities**: Monitoring tools and cluster managers.
2.  **Artisan CLI**: Interactive configuration wizards and dashboards.
3.  **Retro-Modern Apps**: Building beautiful software that runs over SSH.

---

## 🚀 Quick Start

Initialize via CLI:
```bash
rupa create --template terminal
```

### Aesthetic Note
The TUI engine supports **TrueColor** ANSI sequences. Use a compatible terminal for the best experience.
