# Path: Hybrid Interop (Hybrid Adapter) 🤝

The **Hybrid Interop** template is for Artisans who need to blend different runtime environments. It enables seamless communication between native Rupa logic and embedded web content.

---

## 🏗️ What's Inside?

- **Hybrid Adapter**: Pre-configured `rupa-hybrid`.
- **Webview Orchestration**: Tools for embedding and controlling web contexts within native windows.
- **Context Bridge**: A high-performance message bus for sharing reactive state between Native and Web.

---

## 🎯 Best For:

1.  **Progressive Migration**: Gradually moving a legacy web app into the Rupa Native ecosystem.
2.  **Web Widgets**: Embedding complex third-party web content (like charts or maps) inside a Rupa Desktop app.
3.  **Cross-technology Apps**: Projects that require specialized JS libraries alongside high-performance Rust logic.

---

## 🚀 Quick Start

Initialize via CLI:
```bash
rupa create --template hybrid
```
