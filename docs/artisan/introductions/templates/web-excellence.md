# Path: Web Excellence (Web Adapter) 🌐

The **Web Excellence** template brings the Rupa "Artisan" philosophy to the browser. It focuses on performance, SEO, and seamless interactivity using WebAssembly.

---

## 🏗️ What's Inside?

- **Web Adapter**: Pre-configured `rupa-web` with WASM support.
- **Native DOM Integration**: Standard mapping of VNodes to HTML elements.
- **Hybrid Support**: Optional Canvas/WebGPU rendering inside the browser.

---

## 🎯 Best For:

1.  **Interactive Websites**: Content-rich sites that need fine-grained reactivity.
2.  **SaaS Frontends**: Building modern, sleek web dashboards.
3.  **PWAs**: Progressive Web Apps that run everywhere.

---

## 🚀 Quick Start

Initialize via CLI:
```bash
rupa create --template web
```

### Environment Note
Requires the `wasm32-unknown-unknown` target. See [**Installation**](../installation.md) for setup details.
