# Path: Native Power (Desktop Adapter) 🖥️

The **Native Power** template is engineered for high-performance graphical applications. It leverages the full potential of the GPU to deliver buttery-smooth artisan experiences.

---

## 🏗️ What's Inside?

- **Desktop Adapter**: Pre-configured `rupa-desktop` (WGPU + Winit).
- **Agnostic UI**: Includes `rupa-ui` for semantic component building.
- **Window Management**: Default window lifecycle orchestration.

---

## 🎯 Best For:

1.  **GUI Applications**: Professional desktop software (IDE, Design Tools, Dashboards).
2.  **Visual Analytics**: Real-time data visualization with hardware acceleration.
3.  **Artisan Masterpieces**: Apps that require custom shaders and rich typography.

---

## 🚀 Quick Start

Initialize via CLI:
```bash
rupa create --template native
```

### Manifest Configuration
The template automatically sets up the `WGPU` surface and the reactive render loop for you.
