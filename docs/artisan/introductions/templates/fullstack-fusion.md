# Path: Fullstack Fusion (Fullstack Adapter) 🌌

The **Fullstack Fusion** template provides the ultimate unified experience. It combines the power of **Server Authority** with the interactivity of **Native/Web Excellence**.

---

## 🏗️ What's Inside?

- **Fullstack Adapter**: Pre-configured `rupa-fullstack`.
- **Isomorphic Engine**: Built-in support for components that run on both server and client.
- **Universal Hydration**: Automated logic for re-attaching reactive signals to server-rendered HTML.
- **Unified Workspace**: Pre-set directory structure for shared logic, server-only code, and client-only code.

---

## 🎯 Best For:

1.  **Modern SaaS Platforms**: Dashboards that need instant SEO-friendly loading and rich, desktop-like interaction.
2.  **Complex Web Apps**: Large-scale applications where state must be synchronized across the network.
3.  **Universal Masterpieces**: When you want your app to be everywhere, starting from a single, cohesive codebase.

---

## 🚀 Quick Start

Initialize via CLI:
```bash
rupa create --template fullstack
```
