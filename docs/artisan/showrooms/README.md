# Platform Showrooms 🏪

Showrooms are the **Physical Body** of the Rupa Framework. They provide the platform-specific adapters for rendering and interaction.

---

## 🖥️ Target Platforms
Where you can run your Rupa application.

*   **[Desktop](./platform-references.md)**: Hardware-accelerated graphics with WGPU.
*   **[Terminal](./console-references.md)**: Interactive CLI and TUI with Crossterm.
*   **[Web](./platform-references.md)**: WASM and native DOM integration.
