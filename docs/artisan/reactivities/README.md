# Reactivities & Events ⚡

The Rupa Framework uses a fine-grained reactivity engine based on **Signals**, **Memos**, and **Effects**. This allows for surgical UI updates without full-tree re-renders.

---

## 🧬 Reactive Primitives
Core building blocks for state management.

*   **[Signals](./signals.md)**: Observable state containers.
*   **[Memos](./memos.md)**: Derived, cached reactive values.
*   **[Effects](./effects.md)**: Side-effects triggered by reactive changes.
