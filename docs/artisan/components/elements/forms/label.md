# Label 🏷️

A semantic and accessible text element for form fields.

---

## 🏗️ Purpose
`Label` provides the necessary context for an input field. It is designed to work with assistive technologies to ensure that form controls are understandable.

---

## 🚀 Example
```rust
VStack::new()
    .child(Label::new("Email Address"))
    .child(Input::new("you@artisan.rs"))
```
