# Card 📦

A container component with an elevated surface for grouping related information.

---

## 🏗️ Purpose
`Card` is the go-to material for displaying structured content (e.g., product details, user profiles) with a distinct visual boundary.

---

## 🚀 Example
```rust
Card::new()
    .child(Text::new("Card Title").bold())
    .child(Text::new("Supporting body text..."))
```
