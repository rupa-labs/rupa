# Breadcrumb 🍞

A navigation aid that tracks the user's hierarchical location.

---

## 🏗️ Purpose
`Breadcrumb` helps users understand where they are in a deep hierarchy and allows them to quickly navigate back to parent sections.

---

## 🚀 Example
```rust
Breadcrumb::new()
    .path(vec!["Home", "Project", "Settings"])
```
