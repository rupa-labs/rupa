# Tabs 📑

Navigation utility for switching between localized views.

---

## 🏗️ Purpose
`Tabs` organize content into high-level categories that can be swapped without changing the URL or navigating away from the current page.

---

## 🚀 Example
```rust
Tabs::new()
    .tab("Profile", ProfileView)
    .tab("Security", SecurityView)
```
