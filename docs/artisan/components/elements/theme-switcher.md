# Theme Switcher 🌓

A ready-to-use utility for toggling between Light and Dark color modes.

---

## 🏗️ Purpose
`ThemeSwitcher` provides a quick way for Artisans to implement mode-toggling logic in their UI with a single component.

---

## 🚀 Example
```rust
Navbar::new()
    .child(Spacer::new())
    .child(ThemeSwitcher::new())
```
