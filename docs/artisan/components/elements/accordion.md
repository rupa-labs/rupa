# Accordion 🪗

A space-saving component that hides/shows content sections.

---

## 🏗️ Purpose
`Accordion` is used to group content that is optional or secondary, allowing users to expand only what they need to read.

---

## 🚀 Example
```rust
Accordion::new()
    .item("Details", MyDetailsComponent)
    .item("Pricing", MyPricingComponent)
```
