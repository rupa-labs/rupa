# Layout Materials 🏗️

Layout components are the skeletal materials of your application. They provide the spatial structure needed to organize complex interfaces.

---

## 🏗️ Structural Units
*   **[Div & Container](./div.md)**: Bare generic containers.
*   **[VStack](./vstack.md)**: Vertical stacking.
*   **[HStack](./hstack.md)**: Horizontal stacking.
*   **[Row & Col](./row-col.md)**: Semantic aliases for stacks.
*   **[Spacer](./spacer.md)**: Expanding fill unit.
*   **[Section](./section.md)**: Major UI region grouping.
*   **[Divider](./divider.md)**: Visual separator.
*   **[Fieldset](./fieldset.md)**: Bordered container with label.

---

## 📐 Artisan Spacing
Every layout component supports the **Artisan Scaling** system (Xs-Xl6). It is recommended to use these semantic scales instead of raw pixels to maintain a consistent visual rhythm.
