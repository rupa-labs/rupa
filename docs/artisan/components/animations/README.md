# Animation & Motion ✨

Motion is part of the Rupa Framework DNA. It brings your application to life by responding to state changes and lifecycle events with fluid, aesthetic movement.

---

## 🧬 Reactive Motion
Animations in Rupa are not just visual flair; they are reactive side-effects linked to your application's heartbeat. When a `Signal` updates, the motion system ensures the transition to the new state is smooth and natural.

## 🏎️ Performance First
Rupa's motion engine is designed for high efficiency:
- **GPU Accelerated**: Interpolation is handled by specialized shaders.
- **Physics Based**: Optional spring physics for interruptible, realistic movement.
- **Agnostic**: Works identically in native windows and high-end terminals.
